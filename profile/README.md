# Deepnote

Deepnote is a drop-in replacement for Jupyter with AI-first design, sleek UI, new blocks, and native data integrations. Use Deepnote Open Source to work locally in your favorite IDE, then scale to Deepnote Cloud for real-time collaboration, governance, and deployable data apps. Deepnote is used by 500,000 data professionals and companies like Est√©e Lauder, SoundCloud, Statsig, Gusto, and beyond.

## Repositories
- [deepnote](https://github.com/deepnote/deepnote): Main notebook repo

- [jupyterlab-deepnote](https://github.com/deepnote/jupyterlab-deepnote): JupyterLab extension for `.deepnote` projects
- [deepnote-toolkit](https://github.com/deepnote/deepnote-toolkit): Essential Python helpers for Deepnote environments
